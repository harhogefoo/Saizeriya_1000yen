# Saizeriya_1000yen
1000円以内でサイゼリヤメニューの組合わせを出力する

## modules
- docs
- generator
- server

## [docs](./docs)
Github Pagesでサイゼリヤメニューを公開

https://harhogefoo.github.io/Saizeriya_1000yen/menu.json

## [generator](./generator)
SQLiteの内容をJSONに出力する

Travis CIでSQLiteとの自動同期をとる設定を行った。

## [server](./server)
Flask製のサイゼリヤ1000円メニュー出力プロジェクト
